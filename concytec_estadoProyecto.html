⇑ Activo

URI https://purl.org/pe-repo/concytec/estadoProyecto#activo
Definition (es) Utilice para establecer el estado de un proyecto cuando este se encuentra aprobado o en curso.
Preferred label (es) Activo
Preferred label (en) Active

⇑ Suspendido

URI https://purl.org/pe-repo/concytec/estadoProyecto#suspendido
Definition (es) Utilice para establecer el estado de un proyecto cuando este se encuentre interrumpido o anulado.
Preferred label (es) Suspendido
Preferred label (en) Suspended

⇑ Concluido

URI https://purl.org/pe-repo/concytec/estadoProyecto#concluido
Definition (es) Utilice para establecer el estado de un proyecto cuando este se encuentre cerrado, luego de la etapa de cierre operativo o financiero.
Preferred label (es) Concluido
Preferred label (en) Finished
